engine

examples

generated files

/engine/*
/editor/*
/tests/*
/tools/*
/thirdparty/*
/old/*
/experimental/*

huge link time

huge compile time

massive binary

bad incremental builds

add_executable(Game

Never.

import random

def generate_density(seed, size):

    random.seed(seed)

    field = {}

    for x in range(-size, size):
        for y in range(-size, size):
            for z in range(-size, size):

                d = random.random()

                if d > 0.6:
                    field[(x,y,z)] = 1

    return field
